## 2. About
### 기획의도
적은 비율의 한의과에서 과반수 이상의 자동차보험이 청구되고 있으며, 한의과의 명세건수 지속적 증가로 업무가 과중.
연간 2000만건 이상의 심사를 전건 심사하고 있으며, 심사직원 인당 약 하루 800건의 심사가 이루어짐.
현실적으로 모든 기관을 집중 심사하는 것은 불가능 => 중점관리기관 선정 
### 개발목표
심사평가원의 데이터를 이용해 중점관리기관을 선정하여 심사직원에게 알림
